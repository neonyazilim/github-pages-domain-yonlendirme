# github-pages-domain-yonlendirme
İsimtescil tarzı sitelerden nasıl sadece domain alınıp, bu domain github pages ile kullanılır. Siteye girdiğinizde yeni aldığınız domain adınız gözükecek fakat host github pages olacağı için domain ücreti dışında yıllık bir ücret ödemeyeceksiniz. HTTPS sertifikası da olacak. Resimdeki adımlar dışında bir şey yapmanıza gerek yok!
